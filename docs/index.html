Getting Started

New to the Wilderness? You only need 5 commands to start playing. Type !w quickstart in Discord for an interactive guide.

Start → Equip → Venture → Fight → Bank → Repeat

  1. !w start — Create your profile and get starter gear
  2. !w equip starter sword — Equip your weapon and armour
  3. !w shop buy 10 shrimp — Buy food (you auto-eat during fights)
  4. !w venture 5 — Enter the Wilderness at a safe level
  5. !w fight — Fight an NPC and earn loot
  6. !w deposit (or !w depo) — Bank your items so you don't lose them on death

That's it! The game will give you tips as you play. When you're ready, explore Slayer, Crafting, the Grand Exchange, and more.

See the full Getting Started guide or type !w help for all commands.
